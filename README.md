## Краткая теория
Допустим, у вас имеется некая цепь с некой передаточной функцией. Для более простого анализа данной цепи, она разбивается на так называемые звенья. Каждое звено имеет свою заранее известную логарифмическую характеристику и заранее известную передаточную функцию. Деление на звенья происходит довольно простым образом - ПФ цепи раскладывается на множители.

> [!TIP]
> Каждое звено имеет свои амплитудные и фазовые характеристики. Суммарные ЛАХ и ЛФХ получаются при сложении характеристик каждого из звеньев (если звенья стоят последовательно, то есть передаточные функции каждого из звеньев, перемножаются друг на друга и дают передаточную функцию всей цепи).
> Как пример - ПФ цепи K/jw может разложиться на последовательное усилительное (K) и интегрирующее (1/jw) звенья.

Итак, у нас есть 6 типов звеньев:
1. **Усилительное звено** (по факту просто константа)
![Ampl](https://github.com/russkiylis/ZvenoLH/blob/main/images/Ampl.png?raw=true)
2. **Интегрирующее звено** (Линия сверху вниз)
![Integr](https://github.com/russkiylis/ZvenoLH/blob/main/images/Integr.png?raw=true)
3. **Дифференциирующе звено** (Линия снизу вверх)
![Diff](https://github.com/russkiylis/ZvenoLH/blob/main/images/diff.png?raw=true)
4. **Апериодическое звено**  (Сначала 0, затем линия вниз)
![Aperiod](https://github.com/russkiylis/ZvenoLH/blob/main/images/aperiod.png?raw=true)
5. **Форсирующее звено**  (Сначала 0, затем линия вверх)
![Fors](https://github.com/russkiylis/ZvenoLH/blob/main/images/fors.png?raw=true)
6. **Колебательное звено** (по факту, состоит из других звеньев, в программе не реализовано)
![Koleb](https://github.com/russkiylis/ZvenoLH/blob/main/images/koleb.png?raw=true)

> [!NOTE]
> Скорее всего ваша первоначальная задача будет состоять в следующем:
> 1. Разложить ПФ цепи на ПФ звеньев, помноженные друг на друга
> 2. Найти ЛХ каждого из звеньев
> 3. Сложить ЛХ и получить ЛХ цепи

1. Допустим, у тебя есть ПФ в виде дроби: `W(p) = 10(1 + 0.2p) / p(1 + 0.5p)`. Для начала переводим это в формат звеньев. Видим, что тут есть
  - Усилительное звено `10`
  - Интегрирующее звено `1/p`
  - Апериодическое звено `1/(1+0.5p)`
  - Форсирующее звено `(1+0.2p)`.

---
2. Создаём прямо в папке новый [Matlab](https://yandex.ru/search/?text=скачать+матлаб+2025)-файл (Ниже я буду описывать файл [ExampleFromReadme.m](/ExampleFromReadme.m)).
> [!WARNING]
> Новый файл должен быть в корневой папке, то есть в той папке, куда вы скопировали репозиторий, то есть лежать рядом с файлом [ZvenoLH.m](/ZvenoLH.m).

В новом файле расписываем звенья в следующем формате:

```
K = 10;
Ta = 0.5;
Tf = 0.2;

W = {
    struct('Type',"Усил",'K',K),...
    struct('Type',"Интегр"),...
    struct('Type',"Апериод",'T',Ta),...
    struct('Type',"Форс",'T',Tf)
};
```

---
3. Создаём объект, в котором будут созданы логарифмические характеристики:

```
LH = ZvenoLH(W);
```
Теперь у нас есть логарифмические характеристики.

---
4. Время вывести логарифмические характеристики. Это можно сделать вручную, как расписано в [ManualExample.m](/ManualExample.m), если вам нужна большая гибкость в отображении графиков. Если же вам не хочется в этом разбираться, можно вывести суммарную ЛХ цепи одной командой:
```
LH.showSumLH(xlimit, ylimitL, ylimitPhi);  % для вывода ЛХ всей цепи
```
`xlimit` - границы графиков по оси Х. Рекомендую для начала ставить `[1/32 32]`, а затем подгонять.
`ylimitL` - границы графика ЛАХ по оси У. Рекомендую для начала ставить `[-20 20]`.
`ylimitPhi` - границы графика ЛФХ по оси У. Рекомендую для начала ставить `[-150 150]`.

Для вывода ЛХ отдельных звеньев можно воспользоваться этим:
```
LH.showZvenoLH(index, xlimit, ylimitL, ylimitPhi);  % для вывода ЛХ отдельного звена.
```
`index` - порядковый номер звена, например в нашем случае первое звено - усилительное, второе - интегрирующее, и т.д.
